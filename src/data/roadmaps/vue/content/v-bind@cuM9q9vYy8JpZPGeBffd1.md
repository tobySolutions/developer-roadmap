# v-bind

The `v-bind` directive dynamically binds an HTML attribute to data. The shorthand for this directive is `:`

## Example

```html
<script setup>
  import { ref } from 'vue';
  const image_url = ref("path/to/image.png")
</script>

<template>
  <img :src="image_url" />
</template>
```
